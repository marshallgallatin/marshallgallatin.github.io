I'm a little lost about what to write about this week.  We only had a couple classes this week, because we had our first test on Thursday night.  We discussed const pointers and references this week, which is really pretty straight forward once you memorize how the const pointer syntax works.  I thought the test was pretty straight forward, but I'm not a huge fan of tests that expect us to write code by hand.  I feel like its a little artificial, I know I made a mistake on the test, which had I been given a compiler and 3 minutes to write tests, then I wouldn't have made the mistake.  Oh well, thats life.  Nothing on the test was overly difficult, and I think anyone who paid attention should have done well.

The next project was posted today.  I glanced at the page and the code, and I haven't delved deeply into it, but it looks like it is going to be the most difficult project to date.  This time there is not an online judge, so there is no third-party confirmation of correctness.

We also got our grades for project 2 back recently.  I'm happy with my score, but I'm a little conflicted since I gained a few bonus points for my acceptance tests breaking other students' submissions.  On one hand, I really don't mind the competition like aspect, but I wonder about the fairness of it all.  Did the people that my acceptance tests break just get unlucky that my tests happened to run against their code?  They only run our acceptance tests against 5 other students, so there must be some randomization and luck in there.  Or the other possibility is that the students' submissions that my tests were run against were just incorrect to begin with, then it wouldn't have mattered who's acceptance tests were run against them.  I dunno, overall I like the class, and really I kind of like the competition part of it, but I also think our projects should be run against all the acceptance tests rather than a subset of them.  Then again, I'm not sure how the grading would work out.  I'm sure glad that I'm not a teacher and I don't have to figure this out.
